# Resume Entropy

A Chrome Extension that generates tailored resumes, cover letters, and outreach emails using AI.

## Features
*   **Smart Generation**: Analyzes job descriptions to create tailored application documents.
*   **Export**: Download as PDF or DOCX.
*   **Privacy**: Your API key and resume are stored locally on your device.

## How to Install Locally (Free)
You can use this extension completely free by loading it in "Developer Mode".

1.  **Open Chrome Extensions Page**:
    *   Type `chrome://extensions` in your address bar and press Enter.
2.  **Enable Developer Mode**:
    *   Toggle the switch in the top-right corner of the page to **ON**.
3.  **Load Unpacked**:
    *   Click the **"Load unpacked"** button that appears in the top-left.
4.  **Select Folder**:
    *   Navigate to the folder where this project is saved:
    *   `C:\Users\dell\.gemini\antigravity\scratch\job-application-extension`
    *   Click **Select Folder**.

## How to Use
1.  **Setup**:
    *   Click the extension icon (puzzle piece) and pin "Resume Entropy".
    *   Right-click the icon and select **Options**.
    *   Obtain an **OpenAI API Key** or a **Google Gemini API Key** (Free tier available at [aistudio.google.com](https://aistudio.google.com/)).
    *   Enter your API Key and paste your **Base Resume**. Click **Save**.
2.  **Generate**:
    *   Go to any job posting (LinkedIn, Indeed, etc.).
    *   Click the "Resume Entropy" icon to open the side panel.
    *   Select the documents you want (Resume, Cover Letter, Email).
    *   Click **Generate**.
3.  **Export**:
    *   Use the **PDF** or **DOCX** buttons to download your documents.
